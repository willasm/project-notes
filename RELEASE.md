# Release Notes

## [0.0.5] 2022-02-04
### Added
- New command - `Project Notes: Set Globals Notes Folder Location`
- New command - `Project Notes: Open or Create Global Note`
- New command - `Project Notes: Open Existing Global Note (Edit Mode)`
